# ecommerce-site
For AI Task by Leyla Sharifova. \ a1qa

1. AI Tool Used:
ChatGPT

2. Initial Prompt:
"Create a simple Ecommerce homepage"

3. Improved Prompt:
"Write code for a simple ecommerce homepage, include: a header with site a name, navigation links to products, product details, cart, contact pages. a main section with welcome message, footer with dummy copyright, create a visually striking yet useful website code."

4. Why Improved Prompt is better:
The improved prompt is more clear, detailed than just directly saying "create website" etc. It guides AI through what kind of website we want to create and with what path. Unnecessary explanations are just a waste of time. And when specifying we need additionally 5 modules, the AI will understand the command and will do it without changing the topic.

5. Features of Good Prompt:
- Clear
- Well-structured
- Concise
- Focused on output requirements

6. Mistakes found in AI Answers:
In the initial answer, some navigation links, modules were missing and extra things were included. The improved prompt fixed these issues.

7. AI Safety:
Sensitive and personal informations such as passwords, personal data, internal company information, security configurations should NOT be shared with AI. You never know if they're really good encrypted or not and can't trust it.
